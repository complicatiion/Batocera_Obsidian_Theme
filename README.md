# Batocera Obsidian Theme

A clean, bright, and modern **Batocera** theme with light overlays, dark text, soft shadows, rounded corners, and custom wallpaper variants.

## Style

This version reworks the original darker Obsidian look into a lighter and more minimal visual style:

- bright overlays and light backgrounds
- dark, high-contrast text for readability
- soft shadowing and rounded UI panels
- clean font pairing for titles, UI, and system info
- designed for modern setups without losing Batocera performance

## Features

- Light, high-contrast UI
- Rounded panel and overlay styling
- Custom wallpaper set: **MIX 1–5**
- Multiple gamelist background variants
- Optimized for **1080p**
- Scales well to **720p** and **4K**
- Tested with **Batocera v41** and **v42**

## Font Setup

The theme uses the following fonts from the `assets` folder:

- `INT.ttf` → main UI text
- `MONT.ttf` → titles and footer text
- `JET.ttf` → system info

These can be changed directly in `theme.xml` if you want to build your own variant.

## Compatibility

Tested with:

- **Batocera v41**
- **Batocera v42**

Recommended resolution:

- **1080p** preferred
- works well at **720p**
- also suitable for **4K** displays

## Installation

Copy the theme folder to:

```text
/userdata/themes/
```

Then select it in:

**Main Menu → UI Settings → Theme Set**

## Customization

### Wallpapers

The included wallpaper variants use:

- `MIX1`
- `MIX2`
- `MIX3`
- `MIX4`
- `MIX5`

### Gamelist / Theme Assets

You can further customize the look by editing:

- `theme.xml`
- files inside `layouts/`
- image assets inside `assets/`
